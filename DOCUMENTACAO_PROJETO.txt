2. DESCRIÇÃO / RESUMO DO PROJETO
================================================================================

GlobalHoops é uma plataforma web moderna e interativa dedicada exclusivamente 
ao basquetebol, que agrega e apresenta dados em tempo real de jogos, equipas 
e jogadores de diversas ligas profissionais (NBA, WNBA, etc.) através da 
integração com APIs públicas da ESPN.

A aplicação permite aos utilizadores autenticados:
- Acompanhar jogos ao vivo com atualizações automáticas a cada 30 segundos
- Consultar calendários de jogos (próximos e terminados)
- Visualizar estatísticas detalhadas de jogadores e equipas
- Aceder a informações de jogo (lesões, resultados recentes, histórico 
  head-to-head, standings)
- Guardar equipas e jogadores favoritos
- Receber notificações personalizadas
- Consultar notícias relacionadas com basquetebol

Objetivo Principal: 
Centralizar numa única plataforma toda a informação relevante sobre 
basquetebol profissional, com foco em experiência de utilizador premium, 
performance otimizada e design moderno.

  RNF01.5: Timeout de requests API externos: 6 segundos

5. FLUXOS PRINCIPAIS
================================================================================

Fluxo de Autenticação:
  1. Utilizador acede à página inicial (Welcome)
  2. Clica em "Registar" → preenche formulário → conta criada
  3. Login com email/password → redirecionado para Dashboard

Fluxo de Visualização de Jogo:
  1. Dashboard mostra jogos ao vivo/próximos/terminados
  2. Utilizador clica num jogo → GameController::page
  3. Backend chama BasketballApiService::getGameSummary (ESPN API)
  4. Extrai lesões, resultados recentes, head-to-head, standings
  5. Renderiza página Game/Show.jsx com todos os dados
  6. Frontend exibe placar, boxscore, estatísticas, injuries, etc.

Fluxo de Favoritos:
  1. Utilizador visita página de jogador/equipa
  2. Clica em "Adicionar aos Favoritos"
  3. POST /api/favorites → FavoriteController::store
  4. Guarda na tabela user_favorites
  5. Favorito aparece no Dashboard
